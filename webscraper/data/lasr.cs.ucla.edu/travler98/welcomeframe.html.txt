      <td>
      <p> <font size="4">Welcome to the LASR Lab's <a
 href="intro.html"><i><b>Travler</b></i></a> project pages</font> (note
that this link provides a more detailed discussion of the <i>Travler</i>
project, including links to related research, important papers and
software downloads).
The <b>TRA</b>nsparent <b>V</b>irtua<b>L</b> <b>E</b>nvi<b>R</b>onment
(<i>Travler</i>) project was one of several multi-year DARPA-funded
research efforts (1994-1999) within the UCLA Computer Science
Department. This nomadic computing project focused on developing
models, prototyping systems software and conducting experiments to
support the needs of not just multiple users using the same files and
systems, but users that are increasingly mobile.</p>
      <p>The illustration above depicts deployment of the various <i>Travler</i>
project components. Some of the call-outs will link you to a page
describing that component in detail. Although this illustration depicts
      <i>Travler</i> within the dynamics of a military environment, <i>Travler</i>
is equally valuable in any conceivable mobile computing environment. </p>
      <p>Each principal investigator on the <i>Travler</i> project had
an extensive background in a project-essential nomadic computing field:
      </p>

      <p><br>
      <a href="http://www.lk.cs.ucla.edu"><b><i>Leonard
Kleinrock</i></b></a> is recognized as one of the leading experts on
modeling of computer networks. Because of his pioneering work on the
first ARPANET protocols, he has been acknowledged as one of the
"fathers" of the Internet.</p>
      <p><a
 href="http://fmg-www.cs.ucla.edu/ficus-members/popek/index.html"><b><i>Gerald
Popek</i></b></a> has been developing system software for distributed
environments and researching security for system software for over
twenty years.</p>
      <p><a
 href="http://www.cs.ucla.edu/csd/people/faculty_pages/bagrodia.html"><b><i>Rajive
Bagrodia</i></b></a>
is a leading expert in the field of parallel discrete event simulation
and has taken the lead in designing simulation platforms to support
research in mobile computing.</p>
      <p><a href="http://fmg-www.cs.ucla.edu/reiher.new/index2.html"><b><i>Peter
Reiher</i></b></a> has been developing system software for unreliable
distributed environments for fifteen years, including naming systems,
replicated file systems, and systems for secure collaboration over
insecure networks. </p>
