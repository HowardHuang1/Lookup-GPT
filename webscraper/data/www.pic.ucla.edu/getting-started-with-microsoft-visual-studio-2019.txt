                <header class="entry-content-header"></header><div class="entry-content"  itemprop="text" ><p style="text-align: left;">Microsoft Visual Studio is an environment which allows you to create, edit, save, debug, compile, and run C++ programs.</p>
<p><span style="text-decoration: underline;">Projects</span> Every C++ program is contained in its own project directory. Whenever you start a new homework assignment or any C++ program, you must create a project within Visual Studio first and add C++ source code files (text files with extension .cpp) to that project. In this handout, we will show you how.</p>
<p><span style="text-decoration: underline;">Solutions</span> A solution is a file that groups related C++ projects together in order to organize them. You will only need to create two solutions for this course: one for your homework projects and another for example projects given to you in PIC 10A.</p>
<p><span style="text-decoration: underline;">To create your first project and solution for your homework, follow these steps:</span></p>
<p><strong>Step 1</strong>: Open Microsoft Visual Studio by clicking the purple infinity icon <a href="https://www.pic.ucla.edu/wp-content/uploads/2017/10/VS_icon.jpg"><img class="alignnone wp-image-241 size-full" src="https://www.pic.ucla.edu/wp-content/uploads/2017/10/VS_icon.jpg" alt="visualStudioIcon" width="36" height="43" /></a></p>
<p><a href="https://www.pic.ucla.edu/wp-content/uploads/2017/10/VS_2.jpg"><img class="alignnone wp-image-243 size-full" src="https://www.pic.ucla.edu/wp-content/uploads/2017/10/VS_2.jpg" width="615" height="630" /></a></p>
<p>Select “Visual C++” from the drop-down menu. Then press “Start Visual Studio.”</p>
<p><strong><br />
Step 2:</strong> Select “Create a new project.&#8221;</p>
<p><a href="https://www.pic.ucla.edu/wp-content/Temp/VS19-2.jpg"><img class="alignnone size-full wp-image-247" src="https://www.pic.ucla.edu/wp-content/Temp/VS19-2.jpg" alt="step2" width="975" height="535" /></a></p>
<p><strong><br />
Step 3:</strong> Under the &#8220;Language&#8221; drop down menu, select &#8220;C++&#8221; and then select “Empty Project&#8221; then the &#8220;Next&#8221; button.”</p>
<p><a href="https://www.pic.ucla.edu/wp-content/Temp/VS19A.jpg"><img class="alignnone size-full wp-image-254" src="https://www.pic.ucla.edu/wp-content/Temp/VS19A.jpg" alt="step3" width="952" height="559" /></a></p>
<p><a href="https://www.pic.ucla.edu/wp-content/Temp/VS19-3.jpg"><img class="alignnone size-full wp-image-254" src="https://www.pic.ucla.edu/wp-content/Temp/VS19-3.jpg" alt="step3" width="952" height="559" /></a></p>
<p><strong><br />
Step 4:</strong> Configure your new project. Under the &#8220;Project name:&#8221; enter the name of your project “Hw1”.   If you are using your own computer, then one can leave the Location item as is. If you are using a Virtual PIC Lab machine, then under “Location:” enter the location where you want to save your project on the U drive, e.g. “U:\”. Under “Solution name:” enter the name of your solution “Homework”. Then click the “Create” button.</p>
<p><a href="https://www.pic.ucla.edu/wp-content/uploads/2020/04/VS19B_Apporto.jpg"><img class="alignnone wp-image-482 size-large" src="https://www.pic.ucla.edu/wp-content/uploads/2020/04/VS19B_Apporto-1024x586.jpg" alt="" width="1024" height="586" srcset="https://www.pic.ucla.edu/wp-content/uploads/2020/04/VS19B_Apporto-1024x586.jpg 1024w, https://www.pic.ucla.edu/wp-content/uploads/2020/04/VS19B_Apporto-300x172.jpg 300w, https://www.pic.ucla.edu/wp-content/uploads/2020/04/VS19B_Apporto-768x439.jpg 768w, https://www.pic.ucla.edu/wp-content/uploads/2020/04/VS19B_Apporto.jpg 1075w" sizes="(max-width: 1024px) 100vw, 1024px" /></a></p>
<p>Then your project will be ready to start:<br />
<a href="https://www.pic.ucla.edu/wp-content/Temp/VS19C.jpg"><img class="alignnone size-full wp-image-255" src="https://www.pic.ucla.edu/wp-content/Temp/VS19C.jpg" alt="step4" width="708" height="577" /></a></p>
<p><strong><br />
Step 5:</strong> Your solution “Homework” and Project “Hw1” are displayed in the Solution Explorer. Right-click “Source files,” select “Add”, then “New Item” to create source code.</p>
<p><a href="https://www.pic.ucla.edu/wp-content/Temp/VS19D.jpg"><img class="alignnone size-full wp-image-256" src="https://www.pic.ucla.edu/wp-content/Temp/VS19D.jpg" alt="step5" width="975" height="578" /></a></p>
<p><strong><br />
Step 6:</strong> Under Visual C++, select “Code.” Then select “C++” File in the middle frame. Enter the name of your C++ source code file “HelloWorld.cpp” and press “Add.” This will create the source code file (a plain text file) inside your Project1 Project folder and integrate it into your program.</p>
<p><a href="https://www.pic.ucla.edu/wp-content/Temp/VS19E.jpg"><img class="alignnone size-full wp-image-257" src="https://www.pic.ucla.edu/wp-content/Temp/VS19E.jpg" alt="step6" width="975" height="527" /></a></p>
<p><strong><br />
Step 7:</strong> Type your C++ code into the source editor and press CTRL-S to save.</p>
<p><a href="https://www.pic.ucla.edu/wp-content/uploads/2014/09/step7.jpg"><img class="alignnone size-full wp-image-258" src="https://www.pic.ucla.edu/wp-content/uploads/2014/09/step7.jpg" alt="step7" width="914" height="623" srcset="https://www.pic.ucla.edu/wp-content/uploads/2014/09/step7.jpg 914w, https://www.pic.ucla.edu/wp-content/uploads/2014/09/step7-300x204.jpg 300w" sizes="(max-width: 914px) 100vw, 914px" /></a></p>
<p><strong><br />
Step 8:</strong> After editing the source code file, select the “Build” menu and then “Build Hw1” to compile your project. The compiler will translate your source code into a binary executable (.exe) file written in your machine’s native language. The file “Hw1.exe” will be created as a result.</p>
<p><a href="https://www.pic.ucla.edu/wp-content/uploads/2014/09/step8.1.jpg"><img class="alignnone size-full wp-image-259" src="https://www.pic.ucla.edu/wp-content/uploads/2014/09/step8.1.jpg" alt="step8.1" width="783" height="516" srcset="https://www.pic.ucla.edu/wp-content/uploads/2014/09/step8.1.jpg 783w, https://www.pic.ucla.edu/wp-content/uploads/2014/09/step8.1-300x197.jpg 300w" sizes="(max-width: 783px) 100vw, 783px" /></a></p>
<p>You will see the message “Build: 1 Succeeded” in the output window.</p>
<p><a href="https://www.pic.ucla.edu/wp-content/uploads/2014/09/step8.2.jpg"><img class="alignnone size-full wp-image-260" src="https://www.pic.ucla.edu/wp-content/uploads/2014/09/step8.2.jpg" alt="step8.2" width="933" height="581" srcset="https://www.pic.ucla.edu/wp-content/uploads/2014/09/step8.2.jpg 933w, https://www.pic.ucla.edu/wp-content/uploads/2014/09/step8.2-300x186.jpg 300w" sizes="(max-width: 933px) 100vw, 933px" /></a></p>
<p><strong><br />
Step 9:</strong> To run program, select the Debug menu, followed by ”Start without debugging.”</p>
<p><a href="https://www.pic.ucla.edu/wp-content/Temp/VS19I.jpg"><img class="alignnone size-full wp-image-261" src="https://www.pic.ucla.edu/wp-content/Temp/VS19I.jpg" alt="step9" width="975" height="652" /></a></p>
<p>Step 10: The output console will pop up and display the message “Hello World!”</p>
<p><a href="https://www.pic.ucla.edu/wp-content/Temp/VS19J.jpg"><img class="alignnone size-full wp-image-263" src="https://www.pic.ucla.edu/wp-content/Temp/VS19J.jpg" alt="step10" width="558" height="186" /></a></p>
<p>Congratulations! You just completed your first C++ program!</p>
<p>You can find the solution (.sln) file in the solution directory “Z:\Visual Studio Projects\Homework\”</p>
<p><a href="https://www.pic.ucla.edu/wp-content/Temp/VS19K.jpg"><img class="alignnone size-full wp-image-264" src="https://www.pic.ucla.edu/wp-content/Temp/VS19K.jpg" alt="after1" width="810" height="554" /></a></p>
<p>You can find the source code HelloWorld.cpp in your Hw1 project subdirectory:</p>
<p><a href="https://www.pic.ucla.edu/wp-content/Temp/VS19L.jpg"><img class="alignnone size-full wp-image-265" src="https://www.pic.ucla.edu/wp-content/Temp/VS19L.jpg" alt="after2" width="840" height="581" /></a></p>
<p>The object (.obj file) is located in your Project “Hw1” subfolder called “Debug.”</p>
