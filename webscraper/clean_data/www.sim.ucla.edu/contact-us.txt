Sim Requests
If you would like to request services or equipment from the UCLA Simulation Center, please familiarize yourself with our available
Resources
for simulation.
When ready with the required information, submit your request through the Sim Request Portal. You must specify whether you are requesting a Simulation Course, an Equipment Rental, or Standardized Patient.
Submit Course, Equipment Rental, or SP Request
Sim Request Portal
Log in with your Mednet account using multi-factor authentication. For those outside the UCLA system, sign up for an account using the "Sign Up" link in the Request Portal.
Once logged in, select "New SP Request" and fill out the form with the required details.
We recommend using Chrome web browser to access the request form, as other web browsers such as Safari, Firefox and Edge may give error messages.
