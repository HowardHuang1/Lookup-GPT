Remote Access – UCLA Program in Computing
Remote access to your UNIX account is currently supported. In order to access your UNIX account, follow these directions:
Load up your favorite file transfer program
Add a new host to it using SFTP protocol, using port 22
That host’s address should be laguna.pic.ucla.edu
The username should be your UNIX username, and the password should be the password you set up for it
Once connected, you can now upload and download files from your UNIX account
Your web accessible folder is (probably) located in the “
public_html
” folder in the root directory of your account (You can navigate back to your account’s root directory using “
cd ~
“, and into your web accessible folder using “cd public_html/”)
The URL of your web accessible directory is probably “http://www.pic.ucla.edu/~
your-bol-username
“
